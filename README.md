# 💰 Finanças Pessoais

> Organizador financeiro mensal — simples, bonito e que roda direto no navegador, sem instalar nada.

🔗 **[Acesse o site](https://luholiveira56.github.io/financas-pessoais/)**

---

## ✨ Funcionalidades

- 📅 **Navegação por mês** — alterne entre meses e mantenha o histórico separado
- 💵 **Renda mensal** — informe sua renda e acompanhe o percentual comprometido
- 📊 **Dashboard com gráficos** — rosca de distribuição e barras por categoria
- 🗂️ **Categorias personalizáveis** — crie, edite e exclua com cores à sua escolha
- ➕ **Itens ilimitados** por categoria
- ⚠️ **Alerta de orçamento** — aviso automático ao ultrapassar 90% da renda
- 🎯 **Meta de poupança** — sugestão de 20% da renda (regra 50/30/20)
- 📥 **Exportar para CSV** — baixe seus gastos para Excel ou Google Sheets
- 🖨️ **Impressão** — versão otimizada para imprimir
- 🌙 **Modo escuro / claro**
- 💾 **Dados salvos no navegador** — tudo persiste via `localStorage`, sem servidor

---

## 🚀 Como usar

### Online (GitHub Pages)

Acesse diretamente pelo link acima em qualquer navegador, computador ou celular.

Para salvar como atalho no celular:
- **Android (Chrome):** toque nos três pontinhos → *Adicionar à tela inicial*
- **iPhone (Safari):** toque no ícone de compartilhar → *Adicionar à Tela de Início*

### Local

1. Baixe o arquivo `index.html`
2. Abra no navegador (Chrome, Firefox, Safari, Edge)
3. Pronto — sem instalação, sem servidor, sem internet

## 🔒 Privacidade

Todos os dados são salvos **localmente no seu navegador** (`localStorage`). Nenhuma informação é enviada para servidores externos.

> ⚠️ Limpar os dados do navegador ou usar modo anônimo apaga o histórico.
